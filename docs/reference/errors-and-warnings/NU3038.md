# <a name="nuget-warning-nu3038"></a>NuGet 警告 NU3038

### <a name="issue"></a>懸案事項

検証の設定は、リポジトリの副署名を必要としますが、パッケージには、リポジトリの副署名はありません。


### <a name="solution"></a>ソリューション

副リポジトリ パッケージがあるパッケージ ソースを使用します。
