# <a name="nuget-warning-nu1801"></a>NuGet 警告 NU1801

### <a name="issue"></a>懸案事項
フィードを読み取るときにエラーが発生したとき`IgnoreFailedSources`が設定を true に、致命的な警告への変換します。 これのメッセージを含めることが、ジェネリック。

### <a name="solution"></a>ソリューション
有効なソースを指定するための構成を編集します。
