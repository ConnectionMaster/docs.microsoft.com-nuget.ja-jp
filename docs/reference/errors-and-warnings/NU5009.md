# <a name="nuget-error-nu5009"></a>NuGet エラー NU5009
<pre>Cannot find version of msbuild.</pre>

### <a name="issue"></a>懸案事項

NuGet パックの操作には、msbuild を見つけることができません。


### <a name="solution"></a>ソリューション

渡してください、 `-MsBuildPath` NuGet パック コマンド。
