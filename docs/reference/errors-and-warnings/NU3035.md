# <a name="nuget-warning-nu3035"></a>NuGet 警告 NU3035

### <a name="issue"></a>懸案事項

リポジトリの副署名の署名証明書の完全な証明書チェーンを構築できませんでした。


### <a name="solution"></a>ソリューション

問題の原因を検出するタイムスタンプ機関にお問い合わせください。
